Bodegas del sur, Welcome to our API Server

Endpoints

Have fun using any of our endpoints :) See the documentation here

  1. GET all Products 
    https://delsur-api-1.herokuapp.com/products
  2. GET Products by name 
    https://delsur-api-1.herokuapp.com/products?name=malbec
  3. GET Products by category 
    https://delsur-api-1.herokuapp.com/products?category=cervezas
  4. GET Product by id 
    https://delsur-api-1.herokuapp.com/product/1
  5. POST a new Product 
    POST https://delsur-api-1.herokuapp.com/product (cannot test in browser, see an example in our documentation)
  6. UPDATE info of a Product 
    PUT https://delsur-api-1.herokuapp.com/product/update (cannot test in browser, see an example in our documentation)
  7. DELETE a Product 
    DELETE https://delsur-api-1.herokuapp.com/product/delete (cannot test in browser, see an example in our documentation)
  8. GET all Categories 
    https://delsur-api-1.herokuapp.com/categories
  9. POST a new Category 
    POST https://delsur-api-1.herokuapp.com/category (cannot test in browser, see an example in our documentation)
